## Installation

Install via NPM:

```bash
npm install @saiuttej/nestjs-mailer
```

Install via Yarn:

```bash
yarn add @saiuttej/nestjs-mailer
```

Install via PNPM:

```bash
pnpm add @saiuttej/nestjs-mailer
```
